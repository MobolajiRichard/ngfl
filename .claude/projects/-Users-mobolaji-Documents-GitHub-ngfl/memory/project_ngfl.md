ngfl is a fantasy football mobile app for the Nigerian Professional Football League (NPFL).

**Why:** There is no equivalent of FPL for the Nigerian league; the user is building it from scratch.

**How to apply:** Frame all feature suggestions around NPFL clubs, player positions (GK/DEF/MID/FWD), gameweek scoring, transfers, chips, and mini-leagues — the same mechanics as FPL but localised to Nigeria.

Stack: Expo SDK 56 + Expo Router, React Native 0.85, NativeWind v4 (Tailwind), TypeScript strict.
